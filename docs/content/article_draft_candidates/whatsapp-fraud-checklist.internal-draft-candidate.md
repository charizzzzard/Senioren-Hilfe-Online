# Internal Draft Candidate: WhatsApp Fraud Checklist

## 1. Internal Draft Candidate Notice

Dieses Dokument ist ein interner Draft Candidate fuer eine ruhige Checkliste zu verdaechtigen WhatsApp- oder Messenger-Nachrichten.

Es ist kein finaler Artikel. Es ist kein Publish Candidate. Es ist nicht publish-ready und nicht vom Human Operator akzeptiert.

Der Text bereitet eine moegliche spaetere Artikelrichtung vor. Er darf nur intern geprueft werden.

## 2. Intended Reader

Dieser Draft richtet sich an aeltere Leserinnen und Leser, die eine komische Nachricht bekommen haben und nicht sicher sind, wie sie reagieren sollen.

Er richtet sich auch an Angehoerige, die ruhig helfen moechten.

Die Sprache soll erwachsen, freundlich und klar bleiben. Niemand soll sich beschaemt fuehlen, weil eine Nachricht verunsichert.

## 3. Reader Problem

Eine verdaechtige Nachricht kann Druck machen.

Manchmal schreibt angeblich ein Kind, Enkelkind oder eine bekannte Person von einer neuen Nummer. Manchmal geht es schnell um Geld, Hilfe oder einen Link.

Das kann verunsichern. Der wichtigste erste Schritt ist deshalb nicht Tempo, sondern Ruhe.

Diese Checkliste hilft beim ersten Sortieren. Sie kann nicht sicher beweisen, ob eine Nachricht Betrug ist.

## 5. Calm Checklist Draft

Wenn eine Nachricht komisch wirkt, nehmen Sie sich einen Moment.

1. Nicht sofort antworten.
2. Kein Geld senden.
3. Keine Codes, Passwoerter oder Bankdaten weitergeben.
4. Nicht auf Links tippen, wenn Sie unsicher sind.
5. Die angebliche Person ueber einen bekannten Kontaktweg pruefen.
6. Eine vertraute Person um Hilfe bitten, wenn die Nachricht Druck macht.

Diese Schritte sind eine ruhige Orientierung. Sie sind keine Garantie, Betrug sicher zu erkennen.

## 6. Section Draft: First Reaction

Bleiben Sie zuerst ruhig.

Wenn eine Nachricht Sie erschreckt oder unter Druck setzt, legen Sie das Smartphone kurz weg.

Atmen Sie durch. Lesen Sie die Nachricht spaeter noch einmal langsam.

Antworten Sie nicht sofort, wenn Geld, ein Link, ein Code oder eine neue Telefonnummer im Spiel ist. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006] [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

Es ist in Ordnung, erst jemanden zu fragen.

## 7. Section Draft: Warning Signs

Eine Nachricht kann genauer geprueft werden, wenn eines dieser Zeichen vorkommt:

- Jemand behauptet, eine neue Nummer zu haben. [claim: SHO-CLAIM-004 | sources: SHO-SRC-005, SHO-SRC-006]
- Die Person schreibt sehr dringend.
- Es geht schnell um Geld oder eine Ueberweisung. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]
- Sie sollen auf einen Link tippen. [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]
- Die Nachricht klingt anders als die Person sonst schreibt.
- Sie sollen etwas geheim halten.
- Sie sollen einen Code, ein Passwort oder private Daten nennen. [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

Ein einzelnes Warnzeichen beweist keinen Betrug.

Aber ein Warnzeichen reicht, um langsamer zu machen und zu pruefen.

## 8. Section Draft: Known Contact Path Verification

Pruefen Sie die Nachricht ueber einen Kontaktweg, den Sie schon kennen.

Das kann zum Beispiel die alte gespeicherte Telefonnummer sein. Es kann auch ein persoenlicher Anruf oder eine Rueckfrage bei einer vertrauten Person aus der Familie sein. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]

Wichtig ist: Verwenden Sie nicht einfach die neue Nummer aus der Nachricht, wenn Sie unsicher sind.

Fragen Sie ruhig:

- "Hast du mir gerade geschrieben?"
- "Ist das wirklich deine neue Nummer?"
- "Brauchst du wirklich Hilfe oder Geld?"

Wenn Sie keine sichere Rueckmeldung bekommen, warten Sie lieber.

## 9. Section Draft: What Not To Do Too Quickly

Tun Sie diese Dinge nicht zu schnell:

- Geld senden
- einen Link oeffnen
- einen Code weitergeben
- private Daten schicken
- sich hetzen lassen
- allein entscheiden, wenn Sie sich unsicher fuehlen

Wer Druck macht, will oft eine schnelle Reaktion. Eine ruhige Pause ist deshalb ein Schutz.

## 10. Section Draft: How Relatives Can Help Respectfully

Wenn Sie als Angehoerige oder Angehoeriger helfen, bleiben Sie freundlich.

Sagen Sie nicht: "Wie konntest du darauf hereinfallen?"

Besser ist:

"Gut, dass du gefragt hast. Wir pruefen das zusammen."

Helfen Sie beim Sortieren:

- Wer soll geschrieben haben?
- Ist die Nummer bekannt?
- Gibt es Gelddruck?
- Gibt es einen Link?
- Gibt es eine bekannte Nummer, ueber die man sicher nachfragen kann?

Die betroffene Person soll sich ernst genommen fuehlen. Dann fragt sie beim naechsten Mal eher wieder nach.

## 11. Section Draft: When To Seek Official Help

Wenn bereits Geld gesendet wurde, private Daten weitergegeben wurden oder eine Drohung im Raum steht, holen Sie Hilfe.

Moegliche erste Anlaufstellen koennen die eigene Bank, die oertliche Polizei oder eine Verbraucherberatung sein.

Dieser Draft gibt keine Rechtsberatung. Er ersetzt keine Beratung durch zustaendige Stellen.

Wenn Sie unsicher sind, bitten Sie eine vertraute Person, gemeinsam den naechsten sicheren Schritt zu klaeren.
