# Keyword Qualification Records

## Zweck

Keyword Qualification Records sind interne Planungsartefakte. Sie werden aus bestehenden Keyword Discovery Records abgeleitet und ordnen manuelle Keyword-Ideen konservativ nach Suchintention, Seniorentauglichkeit, Evidenzlage, SERP-Status, Monetarisierungsrisiko, internem Linkpotenzial und MVP-Prioritaet ein.

## Explicit Non-Acceptance

- Keyword Qualification Records validieren keine echte Nachfrage.
- Sie behaupten kein Suchvolumen, keine Rankings, keinen Traffic, keine CTR, keine Conversion, keine Revenue, keine Search-Console-Daten, keine Analytics-Daten und kein Nutzerfeedback.
- Sie erstellen keine Cluster Maps.
- Sie erstellen keine SEO Brief Addenda.
- Sie erstellen keine SEO Review Checklists.
- Sie erstellen keine Artikelinhalte.
- Sie erstellen keine Publish Readiness.
- Sie erstellen keine Operator Acceptance.
- Sie schalten keine blockierten Claims frei.

## Brief-spezifische Grenzen

- Brief-002-Qualification-Records muessen erlaubte Claim-Grenzen erhalten und `SHO-CLAIM-007` blockiert halten.
- Brief-002-Qualification-Records duerfen keine WhatsApp block/report UI instructions erzeugen.
- Brief-003-Qualification-Records muessen Screenshot-/Device-Version-Validierung als offen erhalten.
- Brief-003-Qualification-Records duerfen keine geraetespezifische UI-Genauigkeit behaupten.

## Datenstatus

Alle echten Metrikfelder bleiben in diesem Ordner konservativ:

- `serp_observation_status: not_available`
- `competition_or_serp_difficulty_status: not_available`
- `search_volume_status: not_available`
- `ranking_status: not_available`
- `traffic_status: not_available`
- `ctr_status: not_available`
- `conversion_status: not_available`
- `revenue_status: not_available`
- `search_console_status: not_connected`
- `analytics_status: not_connected`
- `user_feedback_status: not_collected`

## Nicht in diesem Ordner erstellt

- Keine Keyword Cluster Maps.
- Keine SEO Brief Addenda.
- Keine SEO Review Checklists.
- Keine Artikel.
- Keine echten Keyword-, Ranking-, Traffic-, CTR-, Conversion-, Revenue- oder Feedbackdaten.
