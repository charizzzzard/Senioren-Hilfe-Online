## Final Article Candidate Text

# Betrugsnachrichten auf WhatsApp erkennen

Manche Nachrichten wirken im ersten Moment harmlos.

Eine vertraute Person scheint zu schreiben. Vielleicht steht dort, dass sie eine neue Nummer hat. Vielleicht klingt die Nachricht dringend. Vielleicht soll schnell Geld überwiesen werden.

In so einer Situation müssen Sie nicht sofort reagieren. Es ist in Ordnung, zuerst innezuhalten. Betrugsnachrichten können verunsichern, und das ist kein Grund, sich zu schämen.

Dieser Artikel hilft Ihnen, typische Warnsignale ruhig zu prüfen. Er gibt keine Garantie, Betrug immer sicher zu erkennen. Er ist auch keine juristische Beratung.

## Das Wichtigste zuerst

Wenn eine Nachricht Sie unter Druck setzt, eine neue Nummer nennt, einen Link enthält oder schnell Geld verlangt, machen Sie langsam. Solche Muster können bei Betrugsnachrichten vorkommen. [claim: SHO-CLAIM-004 | sources: SHO-SRC-005, SHO-SRC-006] [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

Der sicherste erste Schritt ist nicht Eile, sondern Prüfung. Antworten Sie nicht vorschnell. Öffnen Sie keine Links, wenn Sie unsicher sind. Überweisen Sie kein Geld, solange Sie nicht über einen bekannten direkten Kontaktweg geprüft haben, ob die Nachricht wirklich stimmt. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006] [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

Wenn angeblich ein Kind, Enkelkind oder eine andere vertraute Person mit neuer Nummer schreibt, kann das ein Warnsignal sein. Es beweist nicht automatisch Betrug. Aber es ist ein guter Grund, in Ruhe nachzuprüfen. [claim: SHO-CLAIM-004 | sources: SHO-SRC-005, SHO-SRC-006] [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]

## 3-Schritte-Sofort-Check

1. Nicht klicken und nicht zahlen.

   Öffnen Sie keinen Link und senden Sie kein Geld, solange die Nachricht ungeprüft ist. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006] [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

2. Über bekannte Nummer zurückrufen.

   Nutzen Sie eine Telefonnummer, die Sie bereits kennen. Verwenden Sie nicht automatisch die neue Nummer aus der Nachricht. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]

3. Vertrauensperson fragen.

   Wenn Sie unsicher bleiben, zeigen Sie die Nachricht einer vertrauten Person. Das kann ein Angehöriger, eine Nachbarin, ein Freund oder eine andere Person sein, der Sie vertrauen.

## Woran Sie eine Betrugsnachricht erkennen

Eine verdächtige Nachricht muss nicht voller Fehler sein. Manchmal klingt sie sogar freundlich. Achten Sie deshalb vor allem auf das Verhalten, das die Nachricht von Ihnen verlangt.

Ein Warnsignal ist eine neue Nummer. Die Nachricht behauptet vielleicht, die alte Nummer sei nicht mehr erreichbar. Dieses Muster kann bei Messenger-Betrug mit angeblichen Angehörigen vorkommen. [claim: SHO-CLAIM-004 | sources: SHO-SRC-005, SHO-SRC-006]

Ein zweites Warnsignal ist Druck. Sie sollen schnell antworten, schnell klicken oder schnell handeln. Phishing- und Smishing-Nachrichten arbeiten häufig mit Druck, auffälligen Absendern, Links oder Dringlichkeit. [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

Ein drittes Warnsignal ist eine Geldbitte. Wenn eine Nachricht eine Notlage behauptet und zugleich schnell Geld verlangt, prüfen Sie zuerst über einen bekannten Kontaktweg. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]

Auch ein Link kann verdächtig sein. Wenn Sie nicht sicher sind, wer schreibt und wohin der Link führt, öffnen Sie ihn nicht. [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

Keines dieser Zeichen beweist allein mit Sicherheit, dass Betrug vorliegt. Aber jedes dieser Zeichen reicht aus, um nicht sofort zu reagieren.

## Was Sie tun können, wenn Sie unsicher sind

Legen Sie das Smartphone einen Moment weg.

Lesen Sie die Nachricht noch einmal langsam.

Fragen Sie sich:

- Kenne ich diese Nummer wirklich?
- Passt die Sprache zur Person?
- Wird Druck gemacht?
- Soll ich auf einen Link gehen?
- Soll ich Geld senden?
- Kann ich die Person über einen bekannten Weg erreichen?

Wenn Sie unsicher sind, antworten Sie nicht sofort. Nutzen Sie einen bekannten direkten Kontaktweg, zum Beispiel eine alte Nummer oder einen Kontakt, den Sie schon vorher gespeichert hatten. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]

Bewahren Sie die Nachricht nur auf, wenn Sie sie später einer vertrauten Person zeigen möchten. Antworten Sie nicht, klicken Sie nicht und senden Sie kein Geld, solange Sie unsicher sind. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006] [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

Wenn Sie eine akute Gefahr vermuten oder sich stark bedroht fühlen, holen Sie Hilfe über geeignete Notfall- oder Beratungswege. Dieser Artikel beschreibt keine rechtlichen Schritte und ersetzt keine Beratung im Einzelfall.

## Für Angehörige und unterstützende Personen

Wenn Ihnen ein Elternteil, Großelternteil oder eine andere ältere Person eine verdächtige Nachricht zeigt, bleiben Sie ruhig.

Vorwürfe helfen nicht. Viele betrügerische Nachrichten arbeiten mit Unsicherheit, Druck oder Dringlichkeit. [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

Lesen Sie die Nachricht gemeinsam. Fragen Sie freundlich:

- Welche Person soll das sein?
- Gibt es eine bekannte alte Nummer?
- Gibt es einen anderen sicheren Kontaktweg?
- Wurde schnell Geld verlangt?
- Gibt es einen Link oder ungewöhnlichen Druck?

Helfen Sie beim Prüfen über bekannte Kontaktwege, bevor geantwortet oder Geld gesendet wird. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]

Ein guter Satz kann sein:

> Wir schauen uns das in Ruhe zusammen an. Du hast nichts falsch gemacht.

So bleibt die Situation sachlich. Und die betroffene Person fragt beim nächsten Mal eher wieder nach.

## Ruhige Checkliste

Diese Checkliste ist für die interne Review-Fassung gedacht.

- Ich bleibe ruhig.
- Ich antworte nicht sofort.
- Ich klicke keinen Link an, wenn ich unsicher bin. [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]
- Ich überweise kein Geld, solange die Nachricht ungeprüft ist. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]
- Ich rufe über eine bekannte Nummer zurück. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]
- Ich frage eine vertraute Person.
- Ich muss mich nicht schämen, wenn mich eine Nachricht verunsichert.

## Häufige Fehler, die Sie vermeiden können

Der erste Fehler ist Eile.

Viele Menschen möchten sofort helfen, wenn angeblich ein Angehöriger in Not ist. Genau deshalb ist eine Pause so wichtig. Die Pause schützt Sie.

Der zweite Fehler ist Vertrauen in eine neue Nummer.

Eine neue Nummer kann echt sein. Sie kann aber auch Teil eines Betrugsmusters sein. Prüfen Sie deshalb über einen bekannten Weg. [claim: SHO-CLAIM-004 | sources: SHO-SRC-005, SHO-SRC-006] [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]

Der dritte Fehler ist ein schneller Klick.

Links in verdächtigen Nachrichten gehören zu typischen Warnmustern bei Phishing oder Smishing. [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

Der vierte Fehler ist Scham.

Wer verunsichert ist, ist nicht dumm. Betrugsnachrichten sind darauf ausgelegt, Menschen aus dem Gleichgewicht zu bringen. Es ist vernünftig, Hilfe zu holen.
