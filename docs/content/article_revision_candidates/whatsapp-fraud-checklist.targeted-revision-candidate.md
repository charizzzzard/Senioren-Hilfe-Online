## 6. Targeted Revision Candidate Text

# Verdaechtige WhatsApp-Nachricht? Erst pruefen, dann antworten

Eine ungewoehnliche Nachricht kann verunsichern. Vielleicht schreibt angeblich
ein Kind, Enkelkind oder eine bekannte Person von einer neuen Nummer.
Vielleicht geht es ploetzlich um Geld, einen Link oder schnelle Hilfe.

Sie muessen nicht sofort reagieren. Nehmen Sie sich Zeit und pruefen Sie die
Nachricht in Ruhe. Diese Checkliste hilft beim Sortieren. Sie kann nicht sicher
beweisen, ob eine Nachricht Betrug ist.

## Das Wichtigste zuerst

Wenn Sie unsicher sind:

- Antworten Sie nicht sofort.
- Senden Sie kein Geld.
- Geben Sie keine Codes, Passwoerter oder Bankdaten weiter.
- Oeffnen Sie keinen Link, solange Sie bei der Nachricht oder beim Link unsicher sind.

Phishing bedeutet: Jemand versucht, ueber eine Nachricht an Daten,
Passwoerter oder Geld zu kommen. Smishing ist eine Form davon per SMS oder
Messenger-Nachricht.

Druck, Eile, ungewohnte Absender oder Links koennen Warnzeichen sein. Sie sind
aber kein sicherer Beweis fuer Betrug.

## Ruhiger 3-Schritte-Check

### 1. Machen Sie eine Pause

Legen Sie das Smartphone kurz zur Seite. Lesen Sie die Nachricht danach noch
einmal langsam.

Eine Pause gibt Ihnen Zeit, ohne Druck zu entscheiden.

### 2. Pruefen Sie ueber einen bekannten Kontaktweg

Nutzen Sie eine Telefonnummer oder einen Kontaktweg, den Sie bereits kennen.
Verlassen Sie sich bei Unsicherheit nicht nur auf die neue Nummer aus der
Nachricht.

Sie koennen zum Beispiel ruhig fragen:

- "Hast du mir gerade geschrieben?"
- "Ist das wirklich deine neue Nummer?"
- "Brauchst du wirklich Geld oder Hilfe?"

### 3. Holen Sie eine vertraute Person dazu

Wenn Sie weiterhin unsicher sind, zeigen Sie die Nachricht einer Person, der
Sie vertrauen.

Gemeinsam laesst sich oft leichter erkennen, welche Angaben noch geprueft
werden muessen.

## Typische Warnzeichen

Pruefen Sie genauer, wenn:

- eine angeblich bekannte Person mit einer neuen Nummer schreibt
- die Nachricht starken Zeitdruck macht
- schnell Geld oder eine Ueberweisung verlangt wird
- Sie einen Link oeffnen sollen
- Sie einen Code, ein Passwort oder private Daten nennen sollen
- die Nachricht anders klingt als die Person sonst schreibt
- Sie etwas geheim halten sollen

Eine neue Nummer kann echt sein. Auch einzelne andere Warnzeichen beweisen
keinen Betrug. Sie sind ein guter Grund, langsamer zu werden und nachzufragen.

## Was Sie besser nicht sofort tun

Tun Sie diese Dinge nicht unter Zeitdruck:

- Geld senden
- einen Link oeffnen
- Codes oder private Daten weitergeben
- nur ueber die neue Nummer zurueckfragen
- sich zu einer schnellen Entscheidung draengen lassen

Wenn Sie bereits Geld gesendet oder private Daten weitergegeben haben, holen
Sie sich Unterstuetzung. Moegliche erste Anlaufstellen koennen Ihre Bank, die
oertliche Polizei oder eine Verbraucherberatung sein.

Das ist eine allgemeine Orientierung und keine Rechtsberatung.

## Wie Angehoerige helfen koennen

Bleiben Sie freundlich, wenn Ihnen eine aeltere Person eine verdaechtige
Nachricht zeigt.

Vorwuerfe helfen nicht. Ein ruhiger Satz ist besser:

> Gut, dass du gefragt hast. Wir pruefen das zusammen.

Gehen Sie die Nachricht gemeinsam durch:

- Wer soll geschrieben haben?
- Ist die Nummer bekannt?
- Wird Geld verlangt?
- Gibt es einen Link oder einen Code?
- Gibt es einen bekannten Kontaktweg fuer eine Rueckfrage?

Die betroffene Person bleibt an der Entscheidung beteiligt. Angehoerige helfen
als Unterstuetzung, nicht als Kontrolle.

## Kurze Checkliste zum Ausdrucken oder Weitergeben

- Ich bleibe ruhig und antworte nicht sofort.
- Ich sende kein Geld.
- Ich oeffne keinen Link, solange ich unsicher bin.
- Ich gebe keine Codes oder privaten Daten weiter.
- Ich pruefe die Person ueber einen bekannten Kontaktweg.
- Ich frage eine vertraute Person, wenn ich unsicher bleibe.
- Ich weiss: Ein Warnzeichen ist noch kein Beweis.
