# Internal Revision Candidate: WhatsApp Fraud Checklist

## 1. Internal Revision Candidate Notice

Dieses Dokument ist eine interne Revision Candidate Version des WhatsApp-Fraud-Checklist Draft Candidates.

Es ist kein finaler Artikel. Es ist kein Final Article Candidate. Es ist kein Publish Candidate. Es ist nicht publish-ready und nicht vom Human Operator akzeptiert.

Diese Revision verbessert Lesbarkeit, Ruhe, Senior-Reader-Experience und Trust Boundaries. Sie veraendert keine Quellenlage und schaltet keine blockierten Claims frei.

## 2. Intended Reader

Dieser Text richtet sich an aeltere Leserinnen und Leser, die eine ungewoehnliche oder verdaechtige Nachricht bekommen haben.

Er richtet sich auch an Angehoerige, die unterstuetzen moechten, ohne Druck zu machen.

Der Ton soll ruhig, klar und erwachsen bleiben. Niemand soll sich dumm oder schuldig fuehlen, weil eine Nachricht verunsichert.

## 3. Reader Problem

Eine Nachricht kann ploetzlich dringend wirken.

Manchmal schreibt angeblich ein Kind, Enkelkind oder eine bekannte Person von einer neuen Nummer. Manchmal geht es um Geld, einen Link, einen Code oder schnelle Hilfe.

Das kann Stress machen. Gerade dann ist es sinnvoll, kurz anzuhalten.

Diese Checkliste hilft beim ersten Pruefen. Sie kann nicht sicher beweisen, ob eine Nachricht Betrug ist.

## 5. Calm Checklist

Wenn eine Nachricht ungewoehnlich oder verdaechtig wirkt, gehen Sie langsam vor.

1. Nicht sofort antworten.
2. Kein Geld senden.
3. Keine Codes, Passwoerter oder Bankdaten weitergeben.
4. Nicht auf Links tippen, wenn Sie unsicher sind.
5. Die angebliche Person ueber einen bekannten Kontaktweg pruefen.
6. Eine vertraute Person um Hilfe bitten, wenn die Nachricht Druck macht.

Diese Checkliste ist eine ruhige Orientierung. Sie ist keine Garantie, Betrug sicher zu erkennen.

## 6. First Reaction

Bleiben Sie zuerst ruhig.

Wenn eine Nachricht Sie erschreckt oder zur Eile draengt, legen Sie das Smartphone kurz zur Seite.

Lesen Sie die Nachricht spaeter noch einmal langsam.

Antworten Sie nicht sofort, wenn Geld, ein Link, ein Code oder eine neue Telefonnummer im Spiel ist. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006] [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

Es ist in Ordnung, erst zu pruefen und jemanden zu fragen.

## 7. Warning Signs

Eine Nachricht sollte genauer geprueft werden, wenn eines dieser Zeichen vorkommt:

- Jemand behauptet, eine neue Nummer zu haben. [claim: SHO-CLAIM-004 | sources: SHO-SRC-005, SHO-SRC-006]
- Die Nachricht klingt sehr dringend.
- Es geht schnell um Geld oder eine Ueberweisung. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]
- Sie sollen auf einen Link tippen. [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]
- Die Nachricht klingt anders als die Person sonst schreibt.
- Sie sollen etwas geheim halten.
- Sie sollen einen Code, ein Passwort oder private Daten nennen. [claim: SHO-CLAIM-006 | sources: SHO-SRC-007]

Ein Warnzeichen ist kein Beweis.

Aber ein Warnzeichen reicht, um langsamer zu machen und die Nachricht zu pruefen.

## 8. Verify Through Known Contact Path

Pruefen Sie die Nachricht ueber einen Kontaktweg, den Sie schon kennen.

Das kann die alte gespeicherte Telefonnummer sein. Es kann auch ein persoenlicher Anruf oder eine Rueckfrage bei einer vertrauten Person aus der Familie sein. [claim: SHO-CLAIM-005 | sources: SHO-SRC-005, SHO-SRC-006]

Wichtig ist: Verlassen Sie sich nicht nur auf die neue Nummer aus der Nachricht, wenn Sie unsicher sind.

Ruhige Fragen koennen helfen:

- "Hast du mir gerade geschrieben?"
- "Ist das wirklich deine neue Nummer?"
- "Brauchst du wirklich Hilfe oder Geld?"

Wenn Sie keine sichere Rueckmeldung bekommen, warten Sie lieber.

## 9. What Not To Do Too Quickly

Tun Sie diese Dinge nicht zu schnell:

- Geld senden
- einen Link oeffnen
- einen Code weitergeben
- private Daten schicken
- sich hetzen lassen
- allein entscheiden, wenn Sie sich unsicher fuehlen

Wer Druck macht, will oft eine schnelle Reaktion. Eine ruhige Pause ist deshalb ein guter Schutz.

## 10. How Relatives Can Help Respectfully

Wenn Sie als Angehoerige oder Angehoeriger helfen, bleiben Sie freundlich.

Vermeiden Sie Vorwuerfe.

Besser ist:

"Gut, dass du gefragt hast. Wir pruefen das zusammen."

Helfen Sie beim Sortieren:

- Wer soll geschrieben haben?
- Ist die Nummer bekannt?
- Gibt es Gelddruck?
- Gibt es einen Link?
- Gibt es eine bekannte Nummer, ueber die man sicher nachfragen kann?

Die betroffene Person soll sich ernst genommen fuehlen. Dann fragt sie beim naechsten Mal eher wieder nach.

Angehoerige helfen am besten als Unterstuetzung, nicht als Kontrolle.

## 11. When To Seek Official Help

Wenn bereits Geld gesendet wurde, private Daten weitergegeben wurden oder eine Drohung im Raum steht, holen Sie Hilfe.

Moegliche erste Anlaufstellen koennen die eigene Bank, die oertliche Polizei oder eine Verbraucherberatung sein.

Diese Revision gibt keine Rechtsberatung. Sie ersetzt keine Beratung durch zustaendige Stellen.

Wenn Sie unsicher sind, bitten Sie eine vertraute Person, gemeinsam den naechsten sicheren Schritt zu klaeren.
