# Verdächtige WhatsApp-Nachricht? Erst prüfen, dann antworten

Eine ungewohnte Nachricht kann verunsichern. Vielleicht schreibt angeblich ein
Kind, Enkelkind oder eine bekannte Person von einer neuen Nummer. Vielleicht
geht es plötzlich um Geld, einen Link oder schnelle Hilfe.

Sie müssen nicht sofort reagieren. Nehmen Sie sich Zeit. Diese Checkliste hilft
beim Sortieren. Sie kann nicht sicher beweisen, ob eine Nachricht Betrug ist.

## Das Wichtigste zuerst

- Antworten Sie nicht sofort.
- Senden Sie kein Geld.
- Geben Sie keine Codes, Passwörter oder Bankdaten weiter.
- Öffnen Sie keinen Link, solange Sie unsicher sind.

Phishing bedeutet: Jemand versucht, über eine Nachricht an Daten, Passwörter
oder Geld zu kommen. Smishing ist eine Form davon per SMS oder Messenger.
Zeitdruck, ungewohnte Absender oder Links können Warnzeichen sein. Sie sind
kein sicherer Beweis für Betrug.

## Ruhiger 3-Schritte-Check

### 1. Machen Sie eine Pause

Legen Sie das Smartphone kurz zur Seite. Lesen Sie die Nachricht später noch
einmal langsam. Eine Pause gibt Ihnen Zeit, ohne Druck zu entscheiden.

### 2. Prüfen Sie über einen bekannten Kontaktweg

Nutzen Sie eine Telefonnummer oder einen Kontaktweg, den Sie bereits kennen.
Verlassen Sie sich bei Unsicherheit nicht nur auf die neue Nummer aus der
Nachricht.

### 3. Holen Sie eine vertraute Person dazu

Wenn Sie weiter unsicher sind, zeigen Sie die Nachricht einer Person, der Sie
vertrauen. Gemeinsam lassen sich offene Fragen oft ruhiger prüfen.

## Typische Warnzeichen

Prüfen Sie genauer, wenn eine angeblich bekannte Person mit einer neuen Nummer
schreibt, starken Zeitdruck macht, Geld verlangt, einen Link sendet oder nach
Codes, Passwörtern oder privaten Daten fragt.

Eine neue Nummer kann echt sein. Auch ein einzelnes Warnzeichen beweist keinen
Betrug. Es ist ein Grund, langsamer zu werden und nachzufragen.

## Kurze Checkliste zum Ausdrucken oder Weitergeben

- Ich bleibe ruhig und antworte nicht sofort.
- Ich sende kein Geld.
- Ich öffne keinen Link, solange ich unsicher bin.
- Ich gebe keine Codes oder privaten Daten weiter.
- Ich prüfe die Person über einen bekannten Kontaktweg.
- Ich frage eine vertraute Person, wenn ich unsicher bleibe.
- Ich weiß: Ein Warnzeichen ist noch kein Beweis.

## Wenn bereits etwas passiert ist

Antworten Sie nicht unter Druck. Senden Sie kein Geld, keine Codes und keine
privaten Daten. Prüfen Sie über einen bekannten Kontaktweg. Wenn bereits ein
Schaden entstanden sein kann, wenden Sie sich an Ihre Bank, die Polizei oder
eine Verbraucherberatung.

Blockieren oder Melden kann hilfreich sein. Nutzen Sie dafür offizielle App-,
Anbieter- oder Hilfequellen. Diese Checkliste nennt bewusst keine genaue
Bedienfolge in WhatsApp, weil Menüs und Bezeichnungen je nach Gerät und Version
unterschiedlich sein können.
