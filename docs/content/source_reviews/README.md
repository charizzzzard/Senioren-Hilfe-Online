# Manual Source Reviews

## Zweck

Manual Source Review Artefakte dokumentieren manuelle Pruefversuche fuer Quellenkandidaten. Sie sind Traceability-Artefakte zwischen Source Pack, Claim Map und spaeterer Evidence Extraction.

## Statusabgrenzung

| Zustand | Bedeutung |
| --- | --- |
| source candidate | Eine Quelle wurde als Kandidat eingetragen, ist aber nicht verifiziert. |
| manual review attempted | Ein manueller Review-Versuch wurde dokumentiert. |
| line-level evidence available | Konkrete Aussagen oder Zeilen wurden als Evidence dokumentiert. |
| verified source | Eine Quelle wurde durch Operator-/Review-Entscheidung als verifiziert markiert. |
| rejected source | Eine Quelle wurde abgelehnt oder als Duplicate markiert. |

## Regeln

- Ein Manual-Review-Versuch allein verifiziert keine Quelle.
- Official platform source title/reachability ist nicht ausreichend fuer Artikel-Claims.
- WhatsApp/UI/platform sources require line-level evidence and screenshot/device-version review before final instruction drafting.
- Quellen bleiben `candidate`, wenn line-level evidence fehlt.
- Operator Acceptance bleibt erforderlich.

## Explicit Non-Acceptance

Manual Source Reviews sind keine Artikelannahme, keine Publish-Freigabe und keine Operator Acceptance.
