Dieser Ordner enthaelt interne Website-Informationsarchitektur- und Preview-Planungsartefakte fuer Senioren-Hilfe Online.

Die Artefakte sind nicht live. Sie erstellen keine Website-Runtime, veroeffentlichen keine Artikel, setzen keine Publish Readiness, erstellen keine Operator Acceptance, aktivieren keinen Public Launch und verbinden keine Analytics-, Search-Console- oder Feedback-Systeme.
